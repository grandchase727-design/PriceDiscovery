# Appendix A — Methodology & Data Limitations

## What our system provides (used in this report)

- **770-ticker live universe** — daily-refreshed Composite scores, classifications, OER, returns
- **Pre-Momentum 5-agent framework** — forward-looking signal (Micro/Macro/Graph/Catalyst/QVR)

- **Sector-Segmented tier** — top-5 by composite per sector
- **Cross-sectional regime detection** — cyclical/defensive/growth/value tilts + region leadership

## What our system does NOT provide

- ❌ **TAM / market size estimates** — used external knowledge for sizing context; would need IDC/Gartner/SEMI subscriptions for accuracy
- ❌ **Detailed valuation comps** — qualitative ranges based on external context; would need FactSet/Capital IQ for live comps
- ❌ **M&A transaction multiples** — not tracked
- ❌ **Analyst earnings models / forecasts** — QVR component uses revision direction but not actual EPS forecasts
- ❌ **Private companies** — universe is public-only by design

## Caveats on idea generation

1. **All "ProvisionalPM" picks** carry false-positive risk ~15% (historical validation: 14.9% demote within 1 bi-weekly snapshot). Use as half-position with stop-loss.
2. **Sector-segmented picks** under-perform universe-eligible picks at 21d window (+1.62% vs +3.79%). Use for **diversification**, not Sharpe maximization.
3. **OER ≥ 60** indicates statistical overextension, but is **not** a sell signal in isolation. Combined with weakening RSS or classification downgrade = clear exit.
4. **Korea names with WeakQVR** (e.g., Hyundai 005380.KS) — technical strong, fundamentals weak by QVR Y-test. Half-position appropriate.
